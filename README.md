# Overview
A simple, easy-to-use MIB parser for use in clientside SNMP analysis applications.

# Dependencies
* [Typescrpt](https://www.typescriptlang.org/)
* [Node (more specifically: npm)](https://www.npmjs.com/)

# Installation

1. Clone this repo
2. Run `npm install`
3. Run `tsc`
4. Run `npm run main`
